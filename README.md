# Scratch 專案：Untitled-24

這個專案是將 Scratch 的作品轉換為可獨立執行的程式碼版本。

## 專案內容
- `index.html`: 完整功能的網頁版，可直接在瀏覽器執行。
- `main.py`: 使用 Python Turtle 重新實現的邏輯版本。
- `Untitled-24.sb3`: 原始的 Scratch 專案檔案。

## 如何執行
1. **網頁版**：直接點擊 `index.html` 即可。
2. **Python 版**：安裝 Python 後，在終端機輸入 `python main.py`。
